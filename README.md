# Amusement Park Layout (Computer Graphics Project)

## Project Overview

This project is a Computer Graphics (CG) implementation of an amusement park layout using OpenGL (GLUT) in C++.





---

## Technologies Used

* C++
* OpenGL
* GLUT

---

## How to Run (Linux)

### Install dependencies

```bash
sudo apt update
sudo apt install freeglut3-dev
```

### Compile

```bash
g++ park.cpp -o park -lglut -lGL -lGLU
```

### Run

```bash
./park
```

---

## How to Run (Windows - MinGW)

### Compile

```bash
g++ park.cpp -o park -lfreeglut -lopengl32 -lgdi32
```

### Run

```bash
./park
```
